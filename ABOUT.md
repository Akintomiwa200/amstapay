# About AmstaPay

**AmstaPay** is a modern, secure, and fast digital payment platform built for Africa and beyond.

We are making financial transactions simple, accessible, and secure for individuals and businesses across the continent.

## 🎯 Our Mission

To empower every African with seamless, affordable, and secure digital payment solutions — removing barriers to financial inclusion.

## ✨ Key Features

- **Instant P2P Transfers** — Send money to anyone using phone number or QR code
- **Secure Digital Wallet** — Multi-currency support with top-notch security
- **QR Code Payments** — Pay merchants or receive payments instantly
- **Transaction History** — Detailed and searchable payment records
- **Dark/Light Theme** — Beautiful and eye-friendly interface
- **Biometric Authentication** — PIN, Fingerprint & Face ID support
- **Real-time Notifications** — Stay updated on every transaction
- **KYC Integration** — Compliant and secure user verification

## 🛠 Technology Stack

- **Frontend**: React Native + TypeScript
- **State Management**: Zustand + TanStack Query
- **Styling**: NativeWind (Tailwind CSS)
- **Authentication**: Secure JWT + Biometrics
- **Payments**: QR Code + Blockchain-ready architecture
- **Deployment**: Expo / Bare React Native workflow
