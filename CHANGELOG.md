- Secure wallet system with multi-currency support
- QR code payment generation and scanning
- Real-time transaction history
- Send and receive money (P2P)

- Basic KYC flow
- Notification system

### Tech Stack
- React Native + TypeScript
- Expo / Bare workflow
- Zustand + TanStack Query
- NativeWind / Tailwind
- Solana / Blockchain integration ready

- End-to-end encryption for transactions

- Rate limiting and fraud detection hooks
